• Submission instructions

    Please turn in all the files (including the ones your downloaded even if you didn't make changes)

    • There should only be one folder within the tarball that contains all the files.

    • If you have a game scaffold subfolder within your lab (e.g. you cloned from git):

    • mkdir UHusername_labNUM
    • cp *.c *.h game_scaffold/*.c game_scaffold/*.h UHusername_labNUM
    • tar -cvzf UHusername_labNUM.tar.gz UHusername_labNUM
    • rm -rf UHusername_labNUM

    • If all the files are just in your lab folder (i.e. no sub folder), just use the old instructions:

    • mkdir UHusername_labNUM
    • cp *.c *.h UHusername_labNUM
    • tar -cvzf UHusername_labNUM.tar.gz UHusername_labNUM
    • rm -rf UHusername_labNUM
    • Submit the tarbal like usual
